BIN-SOFT Program za obračun zarada

Projekt Objašnjenje

Aplikacija za obračun zarada je jedna u nizu poslovnih aplikacija firme BIN-Soft. Rađena je u Java Spring-boot RESTFull-web-services tehnologiji kao web aplikacija dok je baza podataka MySQL server. Front end aplikacije je rađen u Bootstrap i DHTMLX frameworks.

Projekt Detalji

  • Zarade po satima
  • Krediti
  • Elektronski obrazci
  • Zarade van radnog odnosa
